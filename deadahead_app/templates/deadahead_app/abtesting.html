A/B Testing Tools

A couple of tools to do basic EDA and calculate several p-values on two datasets.

Just paste numeric data into the fields and hit "Go".

This tool does:

  • Plot a combined bloxpot and swarmplot of the provided data.
  • Plot nicely comparable normed histograms and KDEs.
  • Calculate summary statistics.
  • Run a bootstrap hypothesis test with the specified number of permutations (as in this document).
  • Run a Welch t-test using scipy.stats.ttest_ind.
  • Run a Chi-Squared test using scipy.stats.chisquare.
